⚙️ System Configuration
Auto-Refresh Interval
How often the dashboard re-fetches data from the sheet (in seconds). Lower = more up-to-date but more requests. Recommended: 60.
Session Timeout
Auto-logout after this many minutes of inactivity (no mouse or keyboard). Set to 0 to never time out. Recommended: 30.
Invoice Overdue Threshold
Invoices unpaid beyond this many days are flagged as overdue and highlighted in red on the dashboard. Recommended: 15.
Shared App Password
Fallback password for users who don't have a personal password set. Leave blank to keep the current one.
